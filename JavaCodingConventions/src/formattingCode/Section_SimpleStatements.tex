\section{Simple Statements}
Each line may contain at most one single statement. Or, the other way round, each (simple) statement has to be written to its own line.

\keeplisting{7}
Example:
\begin{lstlisting}
// AVOID!!!
++argv; --argc;

// CORRECT
++argv;
--argc;
\end{lstlisting}
